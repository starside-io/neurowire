# @neurowire/web

## 0.5.0

- Add a client-side search box to generated HTML pages (feed pages and per-mesh construct pages): live filter, "N of M shown" count, empty state, debounced, reduced-motion aware. Pages stay fully self-contained.

## 0.4.0

- Add `--today` to the `neurowire-web` CLI: keep only entries published since 00:00 UTC today.

## 0.1.0

- Initial release: `toHtml` and the `neurowire-web` bin that render a feed or mesh into a self-contained HTML news page.
